User Flow Pengujian - Admin

Flow ini fokus untuk Admin operasional. Akses modul mengikuti permission admin, dengan skenario pengujian utama pada streaming, inventaris, publikasi, anggota, dan log.

Role: Admin Cakupan Log: Diri Sendiri + Anggota No Akses: Kelola Data Admin
Flow Super Admin Flow Admin Flow Anggota Hasil Submit

Identitas Penguji

Wajib diisi sebelum menilai flow

Start

7) Form Kuesioner Pengujian

Kartu pertanyaan muncul di setiap kotak flow, submit ada di bawah
Lihat Halaman Hasil
Silakan isi kuesioner berdasarkan hasil pengujian Anda.
Checklist Uji Cepat Admin:
1. Akses modul jalan sesuai skenario operasional.
2. Log aktivitas tidak menampilkan admin lain (kecuali diri sendiri).
3. Export Excel dan Preview PDF bisa dipakai untuk bahan laporan pengujian.
4. Semua tautan flow dapat dibuka manual dari file HTML ini.
