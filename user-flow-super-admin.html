User Flow Pengujian - Super Admin

Flow ini dipakai untuk test keseluruhan fungsi wireframe dari perspektif Super Admin. Semua node bisa dibuka manual dari File Explorer.

Role: Super Admin Cakupan: Semua Modul Tujuan: Skenario Pengujian End-to-End
Flow Super Admin Flow Admin Flow Anggota Hasil Submit

1) Autentikasi & Masuk Dashboard

Start flow
Home / Entry

Masuk dari halaman utama website.

Buka home.html
→
Login

Login dengan akun super admin.

Buka login.html
→
OTP + Reset Password

Skenario recovery akun saat lupa password.

forgot-password.html | otp-verification.html | reset-password.html
→
Dashboard Super Admin

Kontrol utama seluruh sistem.

dashboard.html?role=super_admin

Lihat Halaman Hasil

Checklist Uji Cepat Super Admin:
1. Semua node dapat dibuka dari File Explorer.
2. Flow lintas modul tidak terputus (streaming, inventaris, konten, anggota).
3. Hak akses tertinggi tervalidasi di halaman log aktivitas.
4. Export laporan (CSV/PDF preview) bisa di-trigger saat test.
