Lihat Halaman Hasil
Silakan isi kuesioner berdasarkan hasil pengujian Anda.
Checklist Uji Cepat Anggota:
1. Dashboard anggota tampil lengkap dengan sidebar dan header.
2. Alur tugas streaming berjalan dari request hingga evaluasi.
3. Alur peminjaman anggota lengkap sampai riwayat.
4. Log aktivitas hanya menampilkan data milik anggota sendiri.
