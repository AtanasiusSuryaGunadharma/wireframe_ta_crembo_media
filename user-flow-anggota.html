User Flow Pengujian - Anggota

Flow pengujian sisi anggota dengan fokus pada dashboard anggota (sidebar + header), tugas, monitoring kewajiban, peminjaman, form pendaftaran, notifikasi, dan log diri sendiri.

Role: Anggota Cakupan Log: Diri Sendiri Saja Mode Uji: Fungsionalitas End User
Flow Super Admin Flow Admin Flow Anggota Hasil Submit

Identitas Penguji

Wajib diisi sebelum menilai flow

1) Masuk Sistem Anggota

Start flow
Beranda Publik

Silakan buka home.html terlebih dahulu, lalu klik Login di kanan atas untuk memulai pengujian anggota.

home.html
→
Login Anggota

Setelah berada di halaman login, masukkan username dan password, lalu pilih akun Anggota untuk masuk ke dashboard.

login.html
→
Dashboard Anggota

Setelah login, periksa dashboard anggota dan pastikan sidebar serta header notifikasi tampil dengan benar.

dashboard-anggota.html

2) Profil, Sertifikat, dan Log Pribadi

Identitas anggota
Profil Anggota

Dari dashboard anggota, buka profil untuk melihat atau memperbarui data pribadi sebelum melanjutkan ke fitur lain.

profil-anggota.html
→
Sertifikat Anggota

Setelah profil dicek, lanjutkan ke sertifikat untuk mengunduh sertifikat keanggotaan.

sertifikat-anggota.html
→
Log Aktivitas Saya

Setelah profil dan sertifikat diperiksa, buka log aktivitas pribadi untuk menguji filter dan search hanya pada data sendiri.

log-aktivitas.html?role=anggota

3) Flow Tugas Streaming Anggota

Request sampai evaluasi
Lihat Jadwal Tugas

Setelah masuk sistem, buka jadwal tugas untuk melihat alur tugas misa biasa atau besar yang tersedia.

jadwal-tugas-misa-anggota.html
→
Request Tugas

Setelah jadwal dibuka, ajukan request tugas untuk mengambil jadwal yang masih tersedia.

request-tugas-anggota.html
→
Pembatalan / Tukar Jadwal

Setelah request dipahami, lanjutkan ke pembatalan atau penukaran jadwal bila ada perubahan tugas sebelumnya.

pembatalan-tugas-anggota.html | penukaran-jadwal-tugas-anggota.html
→
Riwayat Tugas + Evaluasi

Setelah alur request dan perubahan jadwal, buka riwayat tugas lalu isi evaluasi streaming untuk menutup alur tugas.

riwayat-tugas-saya.html | evaluasi-streaming-anggota.html

4) Monitoring Kewajiban Tugas

Kontrol kepatuhan pribadi
Monitoring Kewajiban

Setelah alur tugas, buka monitoring kewajiban untuk memantau status kewajiban tugas anggota.

monitoring-kewajiban-tugas-anggota.html

5) Flow Peminjaman Barang

Dari ajukan sampai kembali
Pengajuan Peminjaman

Setelah monitoring kewajiban, lanjutkan ke pengajuan peminjaman barang untuk mengajukan barang yang dibutuhkan.

pengajuan-peminjaman-barang-anggota.html
→
Input Pengambilan

Setelah pengajuan dibuat, lanjutkan ke input pengambilan untuk mengonfirmasi barang yang sudah disetujui.

input-pengambilan-barang-anggota.html
→
Input Pengembalian

Setelah pengambilan, lanjutkan ke input pengembalian untuk menutup transaksi peminjaman barang.

input-pengembalian-barang-anggota.html
→
Riwayat Peminjaman

Setelah pengembalian selesai, buka riwayat peminjaman untuk melihat histori transaksi pribadi secara lengkap.

riwayat-peminjaman-barang-anggota.html

6) Form Pendaftaran Kegiatan + Notifikasi

Partisipasi kegiatan
Daftar Form Kegiatan

Setelah alur barang, buka daftar form kegiatan untuk melihat form pendaftaran yang tersedia.

isi-form-pendaftaran.html
→
Isi Form Detail

Setelah form daftar dibuka, isi form detail dan submit pendaftaran kegiatan sesuai alur sebelumnya.

form-pendaftaran.html | form-pendaftaran-detail.html
→
Notifikasi di Header

Setelah pendaftaran kegiatan, buka dashboard anggota untuk mengecek notifikasi di header beserta filter dan search-nya.

dashboard-anggota.html

7) Selesai Pengujian Anggota

Exit
Kembali ke Home

Setelah seluruh alur anggota selesai, validasi transisi kembali ke beranda untuk menutup sesi pengujian.

home.html?session=active
→
Logout

Setelah kembali ke home, logout untuk mengakhiri sesi pengujian role anggota.

login.html

8) Form Kuesioner Pengujian

Kartu pertanyaan muncul di setiap kotak flow, submit ada di bawah
Lihat Halaman Hasil
Silakan isi kuesioner berdasarkan hasil pengujian Anda.
Checklist Uji Cepat Anggota:
1. Dashboard anggota tampil lengkap dengan sidebar dan header.
2. Alur tugas streaming berjalan dari request hingga evaluasi.
3. Alur peminjaman anggota lengkap sampai riwayat.
4. Log aktivitas hanya menampilkan data milik anggota sendiri.
